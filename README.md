# Playground SQL

![database]

This repository contains code written to demonstrate SQL queries.

| Description        | Command                   |
|:-------------------|:--------------------------|
| Provision database | `./cli.sh up`             |
| Destroy database   | `./cli.sh down`           |
| Run all scripts    | `./cli.sh run`            |
| Run single script  | `./cli.sh run <filename>` |
